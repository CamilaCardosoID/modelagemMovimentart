<h1> Sistema de Banco de Dados - Movimentart</h1>
<h3>Projeto Acadêmico - Faculdade Cruzeiro do Sul</h3>
<p>O projeto tem por finalidade auxiliar na gestão das vendas de artigos de crochê para as artesãs.</p>
<p>01_createTable: CREATE TABLE para criar as tabelas (artesa, cliente, orcamento, produto, pedido)</p>
<p>02_populateTable: INSERT para inserir os dados fictícios nas tabelas.</p>
<p>03_consultas: contém dois tipos de consultas úteis utiizando SELECT.</p>
<p>04_updatedelete: todas as modificações que foram realizadas em caráter de teste usando UPDATE ou DELETE </p>
